Projects

1. Flask-market-app

View on GitHub

Flask Market App is a full-stack web application built using the Flask framework, designed to simulate a small online marketplace environment. The platform allows users to register, log in, and log out securely using Flask-Login for authentication management. Once logged in, users are provided with a virtual wallet (budget) to buy and sell items within the marketplace.

The application features a clean and intuitive user interface where users can browse available products, purchase them if they have enough balance, or list their own items for sale. Each transaction dynamically updates the user’s balance and item ownership in real-time, ensuring accurate synchronization between users and items.


2. Student-management-system

View on GitHub

The Student Management System is a web-based application built using the Django framework, designed to streamline and digitize student-related record management. It provides administrators with a centralized platform to efficiently manage student profiles, academic performance, attendance records, and fee details — all from a single dashboard.

For students, the system offers a secure portal where they can log in to view their marks, attendance status, and fee payments in real time. Administrators can easily update data, generate reports, and monitor student progress without relying on manual record-keeping.


3. Fastapi-blog

View on GitHub

FastAPI Blog is a lightweight yet fully functional RESTful API application developed using the FastAPI framework and SQLAlchemy ORM. The project demonstrates how to build a secure and modular backend for a blogging platform, where users can register, authenticate, and manage blog posts efficiently.

It implements all core CRUD (Create, Read, Update, Delete) operations for both users and posts, showcasing the structure and best practices of modern API development. The application follows a clean modular design using routers, Pydantic schemas, and database models, ensuring scalability, readability, and maintainability.


4. Movie-Recommendation-system

View on GitHub

The Movie Recommendation System is a data-driven Python project designed to provide personalized movie suggestions based on user preferences and viewing history. It leverages collaborative filtering techniques to analyze user ratings and identify patterns of similarity between users and movies.

Using libraries like Pandas and Scikit-learn, the system performs data preprocessing, feature extraction, and similarity computation to deliver accurate and efficient recommendations. The modular codebase ensures scalability and easy experimentation with various recommendation approaches, including user-based and item-based filtering.


5. Job-Portal

View on GitHub

TThe Job Portal is a role-based web application developed using the Flask framework, designed to streamline the recruitment process for both Recruiters and Candidates. It provides a clean and intuitive interface that allows recruiters to post job openings, review applications, and manage listings, while candidates can register, create profiles, and apply for positions that match their skills and interests.

Built with SQLite as the backend database, the platform efficiently stores user data, job listings, and application records, ensuring secure and organized access. Flask’s modular structure and routing system make the app lightweight yet scalable for future enhancements.


6. COVID-19-Visualization

View on GitHub

The COVID-19 Data Visualization project provides an in-depth data analysis and visualization of the global COVID-19 pandemic during 2020. Built entirely using Python, this project leverages Pandas, Matplotlib, and Seaborn to clean, analyze, and visualize country-wise trends of confirmed, recovered, and death cases.

The primary goal of this project is to explore patterns in the spread and impact of COVID-19 through visual storytelling. By analyzing real-world data obtained from Kaggle’s COVID-19 dataset, the system presents insightful charts such as global trend lines, top 10 affected countries, month-wise variations, and correlation heatmaps. The project highlights how data visualization can simplify complex health data, enabling better understanding of global pandemic trends.
